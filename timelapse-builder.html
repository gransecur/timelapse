🎬 Configuração inicial

Para conectar ao Google Drive, você precisa de uma Client ID da API do Google. É gratuito e leva cerca de 5 minutos. Faça isso uma única vez.

Acesse console.cloud.google.com e crie um projeto (pode chamar de "TimeLapse").
No menu lateral, vá em APIs e Serviços → Biblioteca e ative a Google Drive API e a Google Picker API.
Vá em APIs e Serviços → Credenciais → Criar Credenciais → ID do cliente OAuth. Tipo: Aplicativo da Web. Em "origens JS autorizadas" adicione a URL desta página.
Copie o ID do cliente (termina em .apps.googleusercontent.com) e cole abaixo.

Também precisará da API Key do Google para o Picker.
